# Applied-GenAI_LLM-Apps_HR-Assistant-using-RAG
An AI-powered HR assistant designed to answer employee queries by leveraging organizational policy documents. The solution uses document processing, vector search, and conversational AI to deliver accurate, context-aware responses through an interactive chatbot interface.

# Project Summary
In this project, we developed a conversational HR chatbot to improve the efficiency of internal query resolution by leveraging company policy documents. The system was designed to ingest and process HR PDF, extract relevant text and convert it into vector embeddings for efficient retrieval. Using a vector database, the chatbot identifies the most relevant document chunks in response to user queries and generates accurate answers using an OpenAI language model. A structured prompt template was implemented to guide response generation and ensure clarity and consistency. The entire workflow—from document loading and text splitting to embedding generation and question-answering was integrated into a seamless pipeline and exposed through an intuitive user interface built with Gradio. This solution significantly enhances accessibility to HR information, reduces manual effort, and enables faster, more reliable decision-making by employees.
